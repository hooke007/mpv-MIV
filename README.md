# Minimalist Image Viewer

mpv的一种衍生用法，最早只是我个人用来预览waifu2x的输出效果，后来逐渐扩充了各种AI算法；  
而由于mpv的部分良好特性似乎也被很多人用来当作图片查看器，于是合并两类用途有了这个 mpv-lazy 的子项目。
